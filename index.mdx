neu is a deep learning framework built from scratch in Go, using only the Go standard library. It provides everything you need to build, train, and evaluate neural networks — from simple multi-layer perceptrons to LSTM-based language models, sequence-to-sequence models with attention, and deep reinforcement learning agents.

## Get started

<Steps>
  <Step title="Install the module">
    Add neu to your Go module — no external dependencies required.

    ```bash
    go get github.com/itsubaki/neu
    ```
  </Step>
  <Step title="Build a model">
    Compose layers into a Sequential model or use one of the pre-built architectures like MLP, RNNLM, or Seq2Seq.
  </Step>
  <Step title="Train with a Trainer">
    Use `trainer.New(model, optimizer)` and call `Fit` with your dataset, batch size, and number of epochs.
  </Step>
  <Step title="Save and reload">
    Persist trained weights to disk with `model.Save` and restore them with `model.Load`.
  </Step>
</Steps>

<Note>
  neu uses only the Go standard library. There are no C bindings, no GPU dependencies, and no third-party packages to manage.
</Note>
