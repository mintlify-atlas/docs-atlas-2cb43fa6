This guide walks you through installing neu, loading the MNIST dataset, constructing a multilayer perceptron with batch normalization, training it with AdaGrad, and persisting the trained weights — all in plain Go with no external dependencies.

<Steps>

<Step title="Install neu">

Add neu to your Go module. Because neu has no dependencies outside the standard library, this is the only `go get` you need.

```bash
go get github.com/itsubaki/neu
```

Confirm the module entry in your `go.sum` and that your project targets Go 1.22 or later in its `go.mod`:

```go
module your/module

go 1.22
```

<Info>
  If you want to run the full MNIST example from the repository, download the dataset files first with `make mnistdl` from the repo root, which places the raw IDX files under `testdata/`.
</Info>
</Step>

<Step title="Load the MNIST dataset">

neu ships a `dataset/mnist` package that reads the standard IDX files and provides helpers for normalization and one-hot encoding.

```go
import (
    "github.com/itsubaki/neu/dataset/mnist"
    "github.com/itsubaki/neu/math/matrix"
)

// Load returns (TrainDataset, TestDataset, error).
train, test := mnist.Must(mnist.Load("./testdata"))

// Normalize pixel values to [0, 1] and encode labels as one-hot vectors.
x  := matrix.New(mnist.Normalize(train.Image)...) // shape: 60000 × 784
t  := matrix.New(mnist.OneHot(train.Label)...)    // shape: 60000 × 10
xt := matrix.New(mnist.Normalize(test.Image)...)  // shape: 10000 × 784
tt := matrix.New(mnist.OneHot(test.Label)...)     // shape: 10000 × 10
```

`matrix.Matrix` is `[][]float64`. Every operation in neu works on this type, so you can construct or transform data with ordinary Go slice code.
</Step>

<Step title="Build the model">

`model.NewMLP` constructs a fully connected network with batch normalization and ReLU activations between every hidden layer, and a softmax-with-loss output layer.

```go
import (
    "github.com/itsubaki/neu/model"
    "github.com/itsubaki/neu/weight"
)

m := model.NewMLP(&model.MLPConfig{
    InputSize:         mnist.Width * mnist.Height, // 784
    OutputSize:        mnist.Labels,               // 10
    HiddenSize:        []int{50, 50, 50},
    WeightInit:        weight.He,
    BatchNormMomentum: 0.9,
})

// Print a layer-by-layer summary.
fmt.Println(m.Summary()[0])
for i, s := range m.Summary()[1:] {
    fmt.Printf("%2d: %v\n", i, s)
}
```

`weight.He` scales the initial weights by `sqrt(2 / fanIn)`, which is appropriate for ReLU networks. You can substitute `weight.Xavier` or `weight.Glorot` for sigmoid/tanh architectures, or provide your own `func(prevNodeNum int) float64`.

The summary output for this config looks like:

```
*model.MLP
 0: *layer.Affine: W(784, 50), B(1, 50): 39250
 1: *layer.BatchNorm: G(1, 50), B(1, 50): 100
 2: *layer.ReLU
 3: *layer.Affine: W(50, 50), B(1, 50): 2550
 4: *layer.BatchNorm: G(1, 50), B(1, 50): 100
 5: *layer.ReLU
 6: *layer.Affine: W(50, 50), B(1, 50): 2550
 7: *layer.BatchNorm: G(1, 50), B(1, 50): 100
 8: *layer.ReLU
 9: *layer.Affine: W(50, 10), B(1, 10): 510

```
</Step>

<Step title="Configure the optimizer">

neu provides SGD, Momentum, AdaGrad, and Adam. The MNIST example uses AdaGrad with a weight-decay hook:

```go
import (
    "github.com/itsubaki/neu/optimizer"
    "github.com/itsubaki/neu/optimizer/hook"
)

opt := &optimizer.AdaGrad{
    LearningRate: 0.01,
    Hooks: []optimizer.Hook{
        hook.WeightDecay(1e-6),
    },
}
```

To use Adam instead, provide `Alpha`, `Beta1`, and `Beta2`:

```go
opt := &optimizer.Adam{
    Alpha: 0.001,
    Beta1: 0.9,
    Beta2: 0.999,
}
```

<Tip>
  Gradient hooks apply transformations to gradients before the parameter update. `hook.WeightDecay` adds L2 regularization; you can implement your own hook as a `func(params, grads [][]matrix.Matrix) [][]matrix.Matrix`.
</Tip>
</Step>

<Step title="Train with Fit">

`trainer.New` pairs a model with an optimizer. Call `Fit` with your training data, epoch count, batch size, and a `Verbose` callback that receives the current epoch, batch index, and loss after every parameter update.

```go
import (
    "fmt"
    "github.com/itsubaki/neu/trainer"
)

tr := trainer.New(m, opt)

tr.Fit(&trainer.Input{
    Train:      x,
    TrainLabel: t,
    Epochs:     10,
    BatchSize:  100,
    Verbose: func(epoch, j int, loss float64, m trainer.Model) {
        // Print every 10% of an epoch.
        if j%(train.N/100/10) != 0 {
            return
        }

        // Sample a random batch for accuracy reporting.
        mask  := trainer.Random(train.N, 100)
        maskt := trainer.Random(test.N, 100)

        acc  := trainer.Accuracy(m.Predict(matrix.Batch(x, mask)), matrix.Batch(t, mask))
        acct := trainer.Accuracy(m.Predict(matrix.Batch(xt, maskt)), matrix.Batch(tt, maskt))

        fmt.Printf("%3d,%4d: loss=%.4f, train_acc=%.4f, test_acc=%.4f\n",
            epoch, j, loss, acc, acct)
    },
})
```

`Fit` shuffles the full dataset at the start of each epoch using a cryptographically seeded source, so results are stochastic but reproducible if you pass your own `rand.Source` as the second argument.

After 10 epochs on this configuration you can expect test accuracy above 95%.
</Step>

<Step title="Save and load weights">

`model.Save` encodes the model parameters to disk using `encoding/gob`. `model.Load` reads them back. Call `m.SetParams` to restore a checkpoint into any model with the same architecture.

```go
import "github.com/itsubaki/neu/model"

// Save after training.
if err := model.Save("mnist.gob", m.Params()); err != nil {
    log.Fatal(err)
}

// Load in a later run.
params, ok := model.Load("mnist.gob")
if !ok {
    log.Fatal("failed to load params")
}
m.SetParams(params)
```

<Note>
  `model.Load` returns `([][]matrix.Matrix, bool)` rather than an error. The boolean is `false` if the file does not exist or the gob payload cannot be decoded — useful for optional checkpoint resumption at the start of a training run.
</Note>
</Step>

</Steps>

## Complete example

The full MNIST training program from the neu repository combines all of the above steps:

```go
package main

import (
	"flag"
	"fmt"
	"time"

	"github.com/itsubaki/neu/dataset/mnist"
	"github.com/itsubaki/neu/math/matrix"
	"github.com/itsubaki/neu/model"
	"github.com/itsubaki/neu/optimizer"
	"github.com/itsubaki/neu/optimizer/hook"
	"github.com/itsubaki/neu/trainer"
	"github.com/itsubaki/neu/weight"
)

func main() {
	var dir string
	var epochs, hiddenSize, batchSize int
	var momentum, learningRate, lambda float64
	flag.StringVar(&dir, "dir", "./testdata", "")
	flag.IntVar(&epochs, "epochs", 10, "")
	flag.IntVar(&hiddenSize, "hidden-size", 50, "")
	flag.IntVar(&batchSize, "batch-size", 100, "")
	flag.Float64Var(&momentum, "batch-norm-momentum", 0.9, "")
	flag.Float64Var(&learningRate, "learning-rate", 0.01, "")
	flag.Float64Var(&lambda, "weight-decay", 1e-6, "")
	flag.Parse()

	train, test := mnist.Must(mnist.Load(dir))

	x  := matrix.New(mnist.Normalize(train.Image)...)
	t  := matrix.New(mnist.OneHot(train.Label)...)
	xt := matrix.New(mnist.Normalize(test.Image)...)
	tt := matrix.New(mnist.OneHot(test.Label)...)

	m := model.NewMLP(&model.MLPConfig{
		InputSize:         mnist.Width * mnist.Height,
		OutputSize:        mnist.Labels,
		HiddenSize:        []int{hiddenSize, hiddenSize, hiddenSize},
		WeightInit:        weight.He,
		BatchNormMomentum: momentum,
	})

	fmt.Println(m.Summary()[0])
	for i, s := range m.Summary()[1:] {
		fmt.Printf("%2d: %v\n", i, s)
	}
	fmt.Println()

	tr := trainer.New(m, &optimizer.AdaGrad{
		LearningRate: learningRate,
		Hooks:        []optimizer.Hook{hook.WeightDecay(lambda)},
	})

	now := time.Now()
	tr.Fit(&trainer.Input{
		Train:      x,
		TrainLabel: t,
		Epochs:     epochs,
		BatchSize:  batchSize,
		Verbose: func(epoch, j int, loss float64, m trainer.Model) {
			if j%(train.N/batchSize/10) != 0 {
				return
			}

			mask  := trainer.Random(train.N, batchSize)
			xbatch := matrix.Batch(x, mask)
			tbatch := matrix.Batch(t, mask)

			maskt := trainer.Random(test.N, batchSize)
			xtbatch := matrix.Batch(xt, maskt)
			ttbatch := matrix.Batch(tt, maskt)

			acc  := trainer.Accuracy(m.Predict(xbatch), tbatch)
			y    := m.Predict(xtbatch)
			acct := trainer.Accuracy(y, ttbatch)

			fmt.Printf("%3d,%4d: loss=%.4f, train_acc=%.4f, test_acc=%.4f\n",
				epoch, j, loss, acc, acct)
			fmt.Printf("predict: %v\n", y.Argmax()[:20])
			fmt.Printf("label  : %v\n", ttbatch.Argmax()[:20])
			fmt.Println()
		},
	})

	fmt.Printf("elapsed=%v\n", time.Since(now))
}
```

Run it with:

```bash
go run cmd/mnist/main.go --dir ./testdata
```
